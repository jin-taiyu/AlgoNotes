#### STL（Standard Template Library标准模板库）

1. STL list

   - 双向链表

   - 指针访问，插入删除（begin()为第一个迭代器，**end()为最后一个还要后面一个**）

   - 初始化、插入、删除、遍历、查找、释放

     ```c++
     // 初始化
     list<int> l;
     
     // 插入 push_back
     for (int i = 0; i < 10; ++i) {
             l.push_back(i);
     }
     
     // 迭代器
     list<int>::iterator it = l.begin();
     for (list<int>::iterator tmp = l.begin(); tmp != l.end(); tmp++) {
             cout << *tmp << ' ';
     }
     
     // 清空
     for (int i = 0; l.begin() != l.end(); ++i) {
             l.erase(l.begin());
     }
     ```

2. STL queue

   ```c++
   // 初始化
   queue<int> q;
   
   // 入队
   q.push(1);
   
   // 出队
   int tmp = q.front();
   q.pop();
   ```
